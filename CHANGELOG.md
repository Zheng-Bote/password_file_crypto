# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-05-13

### Added
- Transitioned dependency management to Conan v2 (`conan.txt`).
- New CLI flags: `-v` (short version), `--version` (full info), and `--check-update` (GitHub update check).
- Integrated `gh_update_checker` library for automatic version checking.
- Improved Windows support and linker stability.

## [1.0.0] - 2026-03-28

### Added
- Initial release.
